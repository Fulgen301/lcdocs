CNAT - Contact Attachment

In multiple places the engine uses ContactAttachment values (a bitmask) to manage the orientation of objects and processes. For example, a vertex of an object can have the CNAT bit left. If that object has
Bit Value CNAT Description
0 1 CNAT_Left Direction: Left
1 2 CNAT_Right Direction: Right
2 4 CNAT_Top Direction: Up
3 8 CNAT_Bottom Direction: Down
4 16 CNAT_Center Direction: Center (not for attachment)
5 32 CNAT_MultiAttach Special flag: new attachment behaviour for objects with the same direction value at several vertices (from version 4.95.4).
6 64 CNAT_NoCollision Extra flag: non-colliding vertex (from version 4.95.4)
