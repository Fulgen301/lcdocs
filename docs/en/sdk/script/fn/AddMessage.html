AddMessage

Category: Messages
Since engine version: 4.9.6.0 CR

Description

Adds strMessage to a message displayed via Message. The display time is extended according to the length of the added text. Between the previous and the new message a line break is inserted.

Syntax

bool AddMessage (string strMessage);

Parameter

strMessage:
Message that should be added.
See also: CustomMessage, Message, PlayerMessage, PlrMessage
