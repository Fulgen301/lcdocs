GetSkyAdjust

Category: Sky
Since engine version: 4.9.1.0 GWE

Description

Returns the sky modulation as set by SetSkyAdjust. For more information see SetSkyAdjust

Syntax

int GetSkyAdjust (bool fBackColor);

Parameter

fBackColor:
If not 0, the secondary sky color is returned as specified in SetSkyAdjust.

Example

SetSkyAdjust(GetSkyAdjust() & 16777215);
Ensures that the sky is drawn without any transparency (thoroughly covering any background color).
See also: RGB, RGBa, SetGamma, SetMatAdjust, SetSkyAdjust
