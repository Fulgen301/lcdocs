VIS_OverlayOnly

Category: Objects / Visibility
Since engine version: 4.9.7.0 CR

Description

Constant for GetVisibility/SetVisibility. The object is only visible when used as overlay on other objects using SetGraphics. Can't be combined with other VIS_* values.

Syntax

int VIS_OverlayOnly ();

Remark

This value is also defined as a constant. Thus the parameter brackets can be omitted.

Example

var obj;
while (obj=FindObject(0,0,0,0,0,0,0,0,0,obj)) SetVisibility(VIS_All(), obj);
Makes all objects visible.
See also: GetVisibility, SetVisibility, VIS_Allies, VIS_Enemies, VIS_God, VIS_LayerToggle, VIS_Local, VIS_None, VIS_Owner
