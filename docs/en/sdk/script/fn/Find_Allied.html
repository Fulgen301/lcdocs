Find_Allied

Category: Objects / Search
Since engine version: 4.9.6.0 CR

Description

Search criterion: finds all objects owned by friends of the specified player.

Syntax

array Find_Allied (int iPlr);

Parameter

iPlr:
Number of the player to which the found objects must be friendly

Remark

For additional information on the use of this function see FindObjects.
See also: FindObjects
