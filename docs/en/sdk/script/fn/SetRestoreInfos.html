SetRestoreInfos

Category: System
Since engine version: 4.9.10.10 [341] LC

Description

Restores some things in the lobby on restart via the restart button.
The values set in the lobby are restored.
Later ingame changes are not considered.

Syntax

void SetRestoreInfos (int what);

Parameter

what:
Bit mask of the things to be restored:
Name Description
RESTORE_None Nothing. Default.
RESTORE_ScriptPlayers Script players as setup in the lobby, incl. name, team and color.
RESTORE_PlayerTeams Player teams set in the lobby according to player names.
