ClearScheduleCall

Category: Script
Since engine version: 4.9.5.0 CE

Description

Cancels function calls which has been scheduled by ScheduleCall. All scheduled calls to the specified function will be cancelled.

Syntax

bool ClearScheduleCall (object pObj, string szFunction);

Parameters

pObj:
Object for which to cancel scheduled function calls. Can be 0 for the calling object.
szFunction:
Name of the scheduled function.

Example

ClearScheduleCall(FindObject(CLNK), "Redefine");
Cancels all scheduled calls to the function "Redefine" for the first clonk.
See also: Schedule, ScheduleCall
