GetObjectBlitMode

Category: Objects / Display
Since engine version: 4.9.5.0 CE (extended in 4.9.6.0 CR)

Description

Returns the current blit mode of an object.

Syntax

int GetObjectBlitMode (object pObj, int iOverlayID);

Parameters

pObj:
[opt] Object to get the blit mode from. Can be 0 at local call.
iOverlayID:
[opt] If specified, the blit mode of the graphics overlay is returned instead. The overlay must have been previously set using SetGraphics. From CR.

Remarks

For an Overview of the possible blit modes, see SetObjectBlitMode().
For an example, see SetObjectBlitMode().
See also: GetClrModulation, SetObjectBlitMode
