SetViewCursor

Category: Player / View
Since engine version: 4.9.7.0 CR

Description

Sets the player's view to pTargetView.

Syntax

bool SetViewCursor (int iPlr, object pTargetView);

Parameters

iPlr:
Player whose view cursor should be changed.
pTargetView:
Object to which the view should be set to.
See also: SetPlrView
