C4Id

Category: Script / Strings
Since engine version: 4.6.5.0 CP

Description

Converts a string into an id.

Syntax

id C4Id (string szID);

Parameter

szID:
String to be converted into an id.

Example

C4Id("ROCK")
Returns the id 'ROCK'.
jwk, April 2002
