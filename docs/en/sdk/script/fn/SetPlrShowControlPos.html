SetPlrShowControlPos

Category: Player
Since engine version: 4.6.5.0 CP

Description

Enables the visual display of the keyboard control block for a player (siehe SetPlrShowControl).

Syntax

bool SetPlrShowControlPos (int iPlr, int iPos);

Parameters

iPlr:
Player for whom you want to set key display options.
iPos:
Position number. 0 top center (default); 1 bottom right.
See also: SetPlrShowCommand, SetPlrShowControl
