GetMenuSelection

Category: Objects / Menu
Since engine version: 4.9.1.0 GWE

Description

Determines the currently selected menu entry of the currently open menu of an object.

Syntax

int GetMenuSelection (object pObj);

Parameter

pObj:
Object of which to access the menu. Can be 0 in local calls.

Example

Log("Aktuelle Auswahl: Punkt %d...", GetMenuSelection(GetCursor(0));
Writes to the log which menu entry the player has currently selected.
See also: CreateMenu, SetMenuDecoration, SetMenuTextProgress
