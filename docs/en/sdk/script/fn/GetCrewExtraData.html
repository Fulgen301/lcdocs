GetCrewExtraData

Category: Player
Since engine version: 4.9.6.0 CR

Description

Retrieves extra data stored in the player file using SetCrewExtraData.

Syntax

any GetCrewExtraData (object pCrew, string strDataName);

Parameters

pCrew:
Object to get the extra data from. Can be 0 at local call.
strDataName:
Name for the data.

Remark

For details, see SetCrewExtraData.
See also: GetPlrExtraData, SetCrewExtraData
