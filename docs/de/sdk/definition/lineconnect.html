LineConnect

Die LineConnect-Eigenschaft (DWord-Bitmaske) eines Objekts bestimmt, welche Leitungen an das Objekt angeschlossen werden können und welches Leitungsverhalten das Objekt hat.
Bit Wert Name Beschreibung
0 1 C4D_PowerInput Das Ende einer Energieleitung kann angeschlossen werden.
1 2 C4D_PowerOutput Der Anfang einer Energieleitung kann angeschlossen werden.
2 4 C4D_LiquidInput Das Ende eines Rohres kann angeschlossen werden.
3 8 C4D_LiquidOutput Der Anfang eines Rohres kann angeschlossen werden.
4 16 C4D_PowerGenerator Das Objekt ist ein Energieerzeuger.
5 32 C4D_PowerConsumer Das Objekt ist ein Energieverbraucher.
6 64 C4D_LiquidPump Das Objekt ist eine Pumpe.
7 128 Reserviert, wird nicht benutzt.
8 256 C4D_EnergyHolder Das Objekt verliert nicht von allein Energie, auch wenn es kein Lebewesen oder Basis ist.
