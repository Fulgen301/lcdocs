FindObject2

Kategorie: Objekte / Suche
Ab Engineversion: 4.9.6.0 CR

Beschreibung

Gibt ein Objekt zurück, das die angegebenen Suchkriterien erfüllt. Die Suchkriterien werden mittels speziellen Kriteriums-Funktionen der Form "Find_*" angegeben.

Syntax

object FindObject2 (array Criteria,  ...);

Parameter

Criteria:
Suchkriterium in Form einer Find_*-Funktion
...:
[opt] Weitere Suchkriterium in Form einer Find_*-Funktion

Anmerkungen

Anders als bei FindObject findet sich bei objektlokalen Aufrufen von FindObject2 das aufrufende Objekt selbst. Dies kann durch Find_Exclude() verhindert werden.
Die Kriterien werden genau so angegeben wie bei FindObjects und ObjectCount2. Siehe FindObjects für weitere Kriterium-Beispiele.

Beispiel

if(FindObject2(Find_Distance(10), Find_OCF(OCF_CrewMember), Find_Hostile(GetOwner()))
  Explode(20);
Wenn sich ein feindlicher Clonk näher als 10 Pixel an das Objekt heranwagt, explodiert dieses.
