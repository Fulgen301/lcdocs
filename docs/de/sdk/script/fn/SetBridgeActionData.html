SetBridgeActionData

Kategorie: Objekte / Aktivität
Ab Engineversion: 4.9.9.0 CR

Beschreibung

Setzt Zusatzparameter für eine aktuell ausgeführte BRIDGE-Prozedur.

Syntax

bool SetBridgeActionData (int iBridgeLength, bool fMoveClonk, bool fWall, int iBridgeMaterial, object pObj);

Parameter

iBridgeLength:
Länge der Brücke.
fMoveClonk:
Ob sich der Clonk mit der Brücke mitbewegen soll.
fWall:
Sollte true sein, wenn der Clonk keine Brücke, sondern eine Wand (d.h. über seinem Kopf) baut.
iBridgeMaterial:
Materialnummer des Materials aus dem die Brücke bestehen soll.
pObj:
Objekt dessen Aktionsdaten geändert werden sollen. Kann bei lokalen Aufrufen 0 sein.
Siehe auch: GetAction, GetActionTarget, ObjectSetAction, SetAction, SetActionData, SetComDir, SetDir, SetPhase
