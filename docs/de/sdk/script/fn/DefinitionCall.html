DefinitionCall

Kategorie: Script / Funktionsaufruf
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Ruft eine Funktion in einem Script ohne zugehöriges Objekt auf. Der this()-Zeiger gibt in einem solchen Aufruf 0 zurück (Vergleichbar mit dem Szenarioscript).

Syntax

any DefinitionCall (id idDef, string szFunction, any ...);

Parameter

idDef:
ID der Definition, in dessen Script die Funktion aufgerufen wird
szFunction:
Aufzurufende Funktion
...:
[opt] Zusätzliche Parameter, die an die Funktion übergeben werden

Beispiel

DefinitionCall(CPPL, "SetKnowledge", 0);
Gibt Spieler 0 die Pläne für die Burgteile.
Siehe auch: GameCall, ObjectCall
