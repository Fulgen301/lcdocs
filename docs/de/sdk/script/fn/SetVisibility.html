SetVisibility

Kategorie: Objekte / Sichtbarkeit
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Verändert die Sichtbarkeit eines Objekts. Objekte können für beliebige Gruppen sichtbar gemacht werden, indem für dwVisibility eine Bitmaske angegeben wird.

Syntax

bool SetVisibility (int dwVisibility, object pObj);

Parameter

dwVisibility:
Bitmaske, die die neue Sichtbarkeit angibt.
Entweder VIS_All, VIS_OverlayOnly oder VIS_None, oder eine Bitmaske aus VIS_Owner, VIS_Allies, VIS_Enemies, VIS_God und VIS_Local.
pObj:
[opt] Objekt, dessen Sichtbarkeit geändert werden soll. 0 bei lokalem Aufruf.

Beispiele

SetVisibility(VIS_None, GetCursor())
Macht den ausgewählten Clonk von Spieler 0 völlig unsichtbar.
SetVisibility(VIS_Owner | VIS_Allies, GetCursor())
Macht den ausgewählten Clonk von Spieler 0 nur für sich und Verbündete sichtbar.
Siehe auch: GetVisibility, VIS_All, VIS_Allies, VIS_Enemies, VIS_God, VIS_LayerToggle, VIS_Local, VIS_None, VIS_OverlayOnly, VIS_Owner
