ProtectedCall

Kategorie: Script / Funktionsaufruf
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Ruft die geschützte oder öffentliche Funktion szFunction in pObj auf. Es sollte überlegt werden, ob der Aufruf wirklich nötig ist.

Syntax

any ProtectedCall (object pObj, string szFunction, any ...);

Parameter

pObj:
Objekt, in dem die Funktion aufgerufen werden soll.
szFunction:
Funktion, die aufgerufen werden soll.
...:
Parameter der Funktion.
Siehe auch: Call, GameCall, ObjectCall, PrivateCall, eval
