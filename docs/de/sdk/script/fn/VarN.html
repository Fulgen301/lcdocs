VarN

Kategorie: Variablen
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Gibt eine Referenz auf eine funktionslokale Variable zurück. Diese Referenz kann zum Lesen und Schreiben benutzt werden.

Syntax

&VarN (string szName);

Parameter

szName:
Name der Variablen, zu der eine Referenz zurückgegeben werden soll

Beispiel

var v0,v1,v2,v3,v4,v5,v6;
VarN(Format("V%d", Random(7))) = Random(100);
Setzt eine zufällige, threadlokale Variable auf einen zufälligen Wert.
Siehe auch: GlobalN, IsRef, LocalN
