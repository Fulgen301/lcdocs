GetAction

Kategorie: Objekte / Aktivität
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Liefert den Namen der momentanen Aktivität eines Objekts zurück, "Idle", wenn das Objekt keine Aktivität ausführt.

Syntax

string GetAction (object pObj);

Parameter

pObj:
[opt] Abzufragendes Objekt. 0 bei lokalem Aufruf.

Beispiel

if(GetAction(FindObject(CLNK)) eq "Swim") Message("Toll, irgendein Clonk schwimmt");
Siehe auch: GetActTime, GetActionTarget, GetPhase, ObjectSetAction, SetAction, SetActionTargets, SetPhase
