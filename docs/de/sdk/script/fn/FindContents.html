FindContents

Kategorie: Objekte / Suche
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Liefert das erste in einem Behälter enthaltene Objekt vom Typ idType.

Syntax

object FindContents (id idType, object pObj);

Parameter

idType:
Objekttyp, von dem das Inhaltsobjekt sein soll
pObj:
[opt] Durchsuchter Behälter. 0 bei lokalem Aufruf.

Beispiel

var hut;
if (hut=FindObject(HUT2)) Exit(FindContents(CLNK, hut), GetX(hut), GetY(hut));
Befördert einen Clonk aus einer Hütte heraus.
Siehe auch: Contents, FindObject, FindOtherContents
