ArcSin

Kategorie: Arithmetik
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Berechnet den Arcussinus von iVal/iRadius, also Result=arcsin(iVal/iRadius). Dies entspricht Sin(Result)=iVal/iRadius. Die Funktion wird benötigt, um Seitenverhältnisse zu einem Winkel umzurechnen. Für die meisten Fälle ist Angle() aber ausreichend und praktischer.

Syntax

int ArcSin (int iVal, int iRadius);

Parameter

iVal:
Länge der Gegenkathete
iRadius:
Länge der Hypotenuse

Beispiel

Log("Zielwinkel: %d Grad", ArcSin(Abs(GetY(target)-GetY()), ObjectDistance(target)));
Gibt den Zielwinkel vom aufrufenden Objekt zu target aus.
Siehe auch: Angle, ArcCos, Sin
