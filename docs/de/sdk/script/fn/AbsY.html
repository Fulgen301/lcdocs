AbsY

Kategorie: Objekte / Position
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Liefert das relative Y in Bezug auf Objektmitte wenn von einem Objekt aufgerufen. Sonst liefert diese Funktion das übergebene Y zurück

Syntax

int AbsY (int iY);

Parameter

iY:
Y

Beispiel

CreateObject(WIPF, AbsX(100), AbsY(500), -1);
Erzeugt einen Wipf an der Position 100/500 auch wenn in einem Objektscript benutzt.
Siehe auch: AbsX, CreateObject, GetX, GetY
