AddMessage

Kategorie: Nachrichten
Ab Engineversion: 4.9.6.0 CR

Beschreibung

Fügt strMessage zu einer mit Message ausgegebenen Nachricht hinzu. Die Anzeigezeit der Nachricht wird um die de angehängten Nachricht verlängert. Zwischen die originale und die hinzugefügte Nachricht wird ein Zeilenumbruch eingefügt.

Syntax

bool AddMessage (string strMessage);

Parameter

strMessage:
Text, hinzugefügt werden soll.
Siehe auch: CustomMessage, Message, PlayerMessage, PlrMessage
