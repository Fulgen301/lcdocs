Contents

Kategorie: Objekte / Inhalt
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Liefert das indizierte Objekt, das in pObj enthalten ist.

Syntax

object Contents (int iIndex, object pObj);

Parameter

iIndex:
Index des Objekts in pObj, Zählung beginnt mit 0
pObj:
Objekt, in welchem gesucht werden soll; null für lokale Aufrufe
Siehe auch: Contained, FindContents
springer, April 2002
