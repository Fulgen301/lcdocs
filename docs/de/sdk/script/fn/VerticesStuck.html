VerticesStuck

Kategorie: Objekte / Vertices
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Liefert die Anzahl feststeckender Vertices von *pObj

Syntax

int VerticesStuck (object pObj);

Parameter

pObj:
[opt] Zielobjekt

Beispiel

var obj=FindObject(FLAG);
if (VerticesStuck(obj)) RemoveObject(obj);
Entfernt obj wenn einer seiner Vertices feststeckt
Siehe auch: GetVertex, Stuck
