MakeCrewMember

Kategorie: Objekte / Mannschaft
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Fügt ein Objekt (z.B. einen Clonk) der Crew eines Spielers hinzu. Das Objekt muss die CrewMember-Eigenschaft in der DefCore gesetzt haben.
Gibt 1 zurück, wenn das Objekt erfolgreich der Crew hinzugefügt wurde.

Syntax

bool MakeCrewMember (object pObj, int iPlayer);

Parameter

pObj:
Objekt (Clonk), das der Crew hinzugefügt werden soll
iPlayer:
Spieler, dessen Crew das Objekt hinzugefügt werden soll.

Beispiel

var nclnk = CreateObject(CLNK, 0, 0, GetOwner());
MakeCrewMember(nclnk, GetOwner());
Dieser kurze Objektscript erstellt einen neuen Clonk und fügt ihn der Crew seines Besitzers hinzu.
Siehe auch: CrewMember, GetCrew, GrabObjectInfo, OCF_CrewMember, SetCrewStatus
