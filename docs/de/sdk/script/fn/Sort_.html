Sort_*

Die Sort_* - Funktionen dienen dem Sortieren von Suchergebnissen von FindObjects.
Name Beschreibung
Sort_Reverse(array sort) Sortiert entsprechend den in sort angegebenen Sortierspezifikationen aber in umgekehrter Reihenfolge.
Sort_Multiple(array sort, ...) Sortiert nacheinander nach allen angegebenen Sortierspezifikationen, bis die erste keine Gleichheit ergibt.
Sort_Distance(int x, int y) Sortiert nach Entfernung zum (objektlokalen) Punkt x/y.
Sort_Random() Sortiert zufällig.
Sort_Speed() Sortiert nach Geschwindigkeit.
Sort_Mass() Sortiert nach Masse.
Sort_Value() Sortiert nach Objektwert.
Sort_Func(string f, ...) Sortiert nach dem Rückgabewert einer Funktion.
