FlameConsumeMaterial

Kategorie: Landschaft
Ab Engineversion: 4.9.7.0 CR

Beschreibung

Gibt true zurück, wenn das Material an angegebener Position brennbar ist.

Syntax

bool FlameConsumeMaterial (int iX, int iY);

Parameter

iX:
X-Koordinate / X-Offset des zu prüfenden Punktes.
iY:
Y-Koordinate / Y-Offset des zu prüfenden Punktes.

Beispiel

if(FlameConsumeMaterial(GetCursor(0)->GetX(),GetCursor(0)->GetY())) IncinerateLandscape(GetCursor(0)->GetX(),GetCursor(0)->GetY())
Zündet die Landschaft an der Position des gerade ausgewählten Clonks des ersten Spielers an, wenn sie brennbar ist.
Siehe auch: IncinerateLandscape
