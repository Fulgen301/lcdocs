GetLeague

Kategorie: System
Ab Engineversion: 4.9.6.0 CR

Beschreibung

Ist die Liga aktiviert, gibt GetLeague zurück, um welche Liga es sich handelt (z.B. "Melee"). Ist die Liga deaktiviert, gibt GetLeague 0 zurück. Da ein Szenario in mehreren Ligen gleichzeitig sein kann, können mit iIndex zusätzliche Ligen abgefragt werden (https://board.ccan.de/cgi-bin/topic_show.pl?pid=151678#pid151678).

Syntax

string GetLeague (int iIndex);

Parameter

iIndex:
Index der Liga
