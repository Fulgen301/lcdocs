OCF_Living

Kategorie: Objekte / OCF
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Object character flag: Lebewesen. Achtung: Tote Lebewesen sind immer noch OCF_Living. Für lebende Lebewesen gibt es OCF_Alive.

Syntax

int OCF_Living ();

Anmerkungen

Dieser Wert ist als Engine-Konstante definiert. Die Funktionsklammern können daher optional weggelassen werden.
Für weitere Informationen und Beispiele zur Benutzung dieser Konstanten siehe Object character flags.
Siehe auch: FindObject, GetOCF, OCF_Alive, Object character flags
