GBackSolid

Kategorie: Landschaft / Material
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Überprüft ob sich an angegebener Position in der Landschaft feste Materie befindet. Also alles wo Objekte nicht durchkönnen.

Syntax

bool GBackSolid (int iX, int iY);

Parameter

iX:
X-Koordinate, Offset bei lokalem Aufruf
iY:
Y-Koordinate, Offset bei lokalem Aufruf

Beispiel

var obj=FindObject(ROCK);
while(GBackSolid(GetX(obj)-GetX(),GetY(obj)-GetY())) SetPosition(GetX(obj),GetY(obj)-1,obj);
Dieses kleine Stückchen Script versetzt einen Stein an die Oberfläche oder auf den Grund eines Sees.
Siehe auch: GBackLiquid, GBackSemiSolid, GBackSky, GetMaterial
