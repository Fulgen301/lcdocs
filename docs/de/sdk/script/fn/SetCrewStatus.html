SetCrewStatus

Kategorie: Objekte / Mannschaft
Ab Engineversion: 4.9.5.0 CE

Beschreibung

Fügt ein Objekt (z.B. einen Clonk) der Crew eines Spielers hinzu oder entfernt dieses aus der Crew. Das Objekt muss die CrewMember-Eigenschaft in der DefCore gesetzt haben.

Syntax

bool SetCrewStatus (int iPlr, bool fInCrew, object pCrew);

Parameter

iPlr:
Spielernummer des Spielers, dessen Crew verändert werden soll.
fInCrew:
true wenn der Clonk der Crew hinzugefügt und false wenn er aus dieser entfernt werden soll.
pCrew:
CrewMember-Objekt welches hinzugefügt oder entfernt werden soll.

Beispiel

var nclnk = CreateObject(CLNK, 0, 0, GetOwner());
SetCrewStatus(GetOwner(), true, nclnk);
Dieser kurze Objektscript erstellt einen neuen Clonk und fügt ihn der Crew seines Besitzers hinzu.
Siehe auch: CrewMember, GetCrew, GrabObjectInfo, MakeCrewMember, OCF_CrewMember
