COMD_Stop

Kategorie: Objekte / Bewegung
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Bewegungsrichtung: Anhalten. Für weitere Hinweise siehe SetComDir.

Syntax

int COMD_Stop ();

Anmerkungen

Dieser Wert ist als Engine-Konstante definiert. Die Funktionsklammern können daher optional weggelassen werden.
COMD_None und COMD_Stop sind identisch und haben den Wert 0.

Beispiel

SetComDir(COMD_None(), GetCursor());
Lässt den aktuell ausgewählten Clonk des ersten Spielers anhalten.
Siehe auch: COMD_None, GetComDir, SetComDir
