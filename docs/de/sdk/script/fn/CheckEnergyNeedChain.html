CheckEnergyNeedChain

Kategorie: Objekte / Status
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Überprüft, ob ein an das angegebene Objekt angeschlossener Verbraucher Energie benötigt. Dabei werden auch indirekt (also über mehrere Leitungen über einen weiteren Verbraucher) angeschlossene Verbraucher überprüft.

Syntax

bool CheckEnergyNeedChain (object pObj);

Parameter

pObj:
[opt] Objekt, von dem aus gesucht werden soll. 0 bei lokalem Aufruf.

Anmerkung

Es wird nur in die Richtung geprüft, in der auch Energie durch Energieleitungen fließen kann. Also vom Versorger zum Verbraucher.

Beispiel

if(CheckEnergyNeedChain()) SetAction("Burning");
Script für ein Kraftwerk: Wenn einer der angeschlossenen Verbraucher Energie benötigt, wird der Ofen angeheizt.
Siehe auch: DoEnergy, EnergyCheck, GetEnergy
