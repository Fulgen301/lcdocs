RemoveVertex

Kategorie: Objekte / Vertices
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Entfernt einen Vertex von einem Objekt.

Syntax

int RemoveVertex (int iX, object pObj);

Parameter

iX:
Index des Vertex, das entfernt werden soll.
pObj:
[opt] Objekt, von dem ein Vertex entfernt werden soll. 0 bei lokalem Aufruf.

Anmerkung

Achtung: Bei jeder Vertexaktualisierung (also beispielsweise dann, wenn sich Fertigstellungswert oder Rotation eines Objekts ändern) werden alle Vertices wieder auf die Definitionswerte zurückgesetzt.

Beispiel

while(RemoveVertex()) ;
Entfernt alle Vertices vom aufrufenden Objekt.
Siehe auch: AddVertex, GetVertex, GetVertexNum, SetVertex
