Music

Kategorie: Musik
Ab Engineversion: 4.6.5.0 CP (erweitert ab 4.9.1.0 GWE)

Beschreibung

Spielt ein Musikstück ab. Die entsprechende Datei muss in einer der aktiven Musikdateien vorhanden sein. Sind im ausgewählten Szenario Musikdateien vorhanden, gilt die Szenariodatei als aktive Musikdatei, andernfalls wird die globale Musikdatei Music.c4g genutzt. Bei erfolgreichem Aufruf dieser Funktion wird die Musikwiedergabe aktiviert, auch wenn sie in den Optionen des Menüsystems deaktiviert wurde.

Syntax

void Music (string szSongName, bool fLoop);

Parameter

szSongName:
[opt] Dateiame des Musikstückes. Die Endung muss mit angegeben werden. Wird der Parameter weggelassen oder 0 übergeben, so wird die Musikwiedergabe deaktiviert (auch wenn sie in den Optionen aktiviert wurde)
fLoop:
[opt] Wenn 1, wird das angegebene Musikstück andauernd abgespielt.
Siehe auch: MusicLevel, SetPlayList, Sound
