Sin

Kategorie: Arithmetik
Ab Engineversion: 4.6.5.0 CP (erweitert ab 4.9.5.0 CE)

Beschreibung

Liefert sin(iAngle) * iRadius.

Syntax

int Sin (int iAngle, int iRadius, int iPrec);

Parameter

iAngle:
Winkel in Grad
iRadius:
Radius
iPrec:
[opt] Der angegebene Winkel wird, wenn angegeben, vor der Berechnung durch diesen Wert dividiert. Bei einem Winkel von 455 und einer Präzision von 10 würde intern also mit einem Winkel von 45,5° gerechnet werden.
Siehe auch: Angle, Cos
