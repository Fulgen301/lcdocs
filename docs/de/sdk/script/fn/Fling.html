Fling

Kategorie: Objekte / Bewegung
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Verschleudert pObj in die angegebene Richtung. Diese Funktion ist für Lebewesen konzipiert, da sie die entsprechenden Aktivitäten setzt. Sie löscht zusätzlich den Killer des Objektes.

Syntax

bool Fling (object pObj, int iXDir, int iYDir, int iPrec, bool fAddSpeed);

Parameter

pObj:
Objekt, das weggeschleudert wird. Muss angegeben werden.
iXDir:
X-Schleudergeschwindigkeit
iYDir:
Y-Schleudergeschwindigkeit
iPrec:
[opt] Genauigkeit. Keine Angabe oder 0 entspricht 1.
fAddSpeed:
[opt] Wenn true, wird die Geschwindigkeit des Objektes hinzuaddiert.

Beispiel

Fling(GetCursor(0), 10, -15);
Verschleudert den ausgewählten Clonk nach oben rechts.
Siehe auch: Jump, SetXDir, SetYDir
