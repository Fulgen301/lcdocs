GetPlayerInfoCoreVal

Kategorie: Get*Val
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Gibt einen Eigenschaftswert aus der Player.txt eines Spielers zurück.

Syntax

any GetPlayerInfoCoreVal (string strEntry, string strSection, id iPlayer, int iEntryNr);

Parameter

strEntry:
Name des abzufragenden Eintrags
strSection:
[opt] Sektion des abzufragenden Eintrags. Wird 0 übergeben, werden alle Sektionen nach einem Eintrag mit dem entspr. Namen durchsucht.
iPlayer:
[opt] Spieler, dessen Eintrag abgefragt werden soll.
iEntryNr:
[opt] Nummer des abzufragenden Wertes, wenn der Eintrag mehrere Werte enthält.

Anmerkung

Es kann aus technischen Gründen keine analoge Funktion SetPlayerVal geben.
Siehe auch: GetActMapVal, GetObjectInfoCoreVal, GetObjectVal, GetPlayerVal, GetScenarioVal
