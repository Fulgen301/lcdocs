Abs

Kategorie: Arithmetik
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Bildet den Absolutwert (Betrag), entfernt also das Vorzeichen des ersten Parameters.

Syntax

int Abs (int iValue);

Parameter

iValue:
Wert, von dem der absolutwert gebildet werden soll

Beispiel

Log("Aktuelle Fluggeschwindigkeit: %d Clonks/min", Abs(GetXDir(FindObject(BLMP))));
Gibt die Fluggeschwindigkeit eines Luftschiffes aus.
Sven2, November 2001
