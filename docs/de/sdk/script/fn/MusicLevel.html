MusicLevel

Kategorie: Musik
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Setzt die Lautstärke, mit der die Hintergrundmusik abgespielt wird (siehe Music).

Syntax

int MusicLevel (int iLevel);

Parameter

iLevel:
Gewünschte Lautstärke (0-100).

Beispiel

local iVolume;

func Initialize()
{
  iVolume = 100;
}

func Timer()
{
  iVolume--;
  MusicLevel(iVolume);
  if(iVolume <= 0)
  {
    Music();
    RemoveObject();
  }
}
Script eines Musik-Fadeout-Objekts. Die Musiklautstärke wird in der Funktion "Timer" bei jedem Aufruf ein Stück reduziert, bis die Lautstärke 0 ist. Dann wird die Musik deaktiviert und das Objekt gelöscht.
Siehe auch: Music
