VIS_Allies

Kategorie: Objekte / Sichtbarkeit
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Konstante für GetVisibility/SetVisibility. Das Objekt ist für alle Verbündeten des Besitzers sichtbar.

Syntax

int VIS_Allies ();

Anmerkung

Dieser Wert ist als Engine-Konstante definiert. Die Funktionsklammern können daher optional weggelassen werden.

Beispiel

var obj;
while (obj=FindObject(0,0,0,0,0,OCF_CrewMember,0,0,0,obj)) SetVisibility(VIS_Owner() | VIS_Allies(), obj);
Macht alle gesteuerten Clonks nur noch für ihre Besitzer und deren Verbündete sichtbar.
Siehe auch: GetVisibility, SetVisibility, VIS_All, VIS_Enemies, VIS_God, VIS_LayerToggle, VIS_Local, VIS_None, VIS_OverlayOnly, VIS_Owner
