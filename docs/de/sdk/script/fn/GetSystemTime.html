GetSystemTime

Kategorie: System
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Gibt die momentane Uhrzeit zurück.

Syntax

int GetSystemTime (int iWhat);

Parameter

iWhat:
Was genau zurückgegeben werden soll (siehe Anmerkung)

Anmerkungen

Je nach Wert von iWhat gibt GetSystemTime die folgende Angabe zurück:
iWhat Rückgabe
0 Jahr
1 Monat (1 = Januar, 2 = Februar usw.)
2 Wochentag (0 = Sonntag, 1 = Montag usw.)
3 Tag
4 Stunde
5 Minute
6 Sekunde
7 Millisekunde
Da die Zeiten bei unterschiedlichen Computern variieren können, gibt diese Funktion, um eine Syncunsicherheit zu vermeiden, beim Netzwerkspiel und bei Filmaufnahmen 0 zurück.

Beispiel

func SageUhrzeit()
{
	Message("Die momentane Uhrzeit ist: %02d:%02d:%02d", this(), GetSystemTime(4), GetSystemTime(5), GetSystemTime(6));
}
