SetComDir

Kategorie: Objekte / Bewegung
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Setzt die Befehlsrichtung eines Objekts (siehe GetComDir)

Syntax

int SetComDir (int iComDir, object pObj);

Parameter

iComDir:
Befehlsrichtung. Muss COMD_None(), COMD_Down(), COMD_DownLeft(), COMD_DownRight(), COMD_Left(), COMD_Right(), COMD_Up(), COMD_UpLeft() oder COMD_UpRight() sein.
pObj:
[opt] Objekt, dessen Befehlsrichtung gesetzt werden soll. 0 bei lokalem Aufruf.

Beispiel

var pClnk = GetHiRank(0);
SetComDir(COMD_None(), pClnk);
SetCommand(pClnk, "");
Gibt dem höchstrangigsten Clonk von Spieler 1 den Befehl, stehen zu bleiben. Vorhandene Kommandos werden gelöscht.
var pClnk = GetHiRank(0);
ObjectSetAction(pClnk, "Dig");
SetComDir(COMD_DownRight());
Lässt den höchstrangigsten Clonk von Spieler 1 nach rechts unten graben.
Siehe auch: GetComDir, SetDir
