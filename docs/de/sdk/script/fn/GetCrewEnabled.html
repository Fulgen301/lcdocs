GetCrewEnabled

Kategorie: Objekte / Mannschaft
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Fragt ab, ob ein Mannschaftsmitglied gegenwärtig angewählt werden kann.

Syntax

bool GetCrewEnabled (object pObj);

Parameter

pObj:
Objekt, dessen Status abgefragt werden soll. 0 bei lokalem Aufruf.

Beispiel

if (!GetCrewEnabled(GetHiRank())) SetCrewEnabled(1, GetCursor());
Aktiviert den höchstrangigen Clonk des ersten Spielers, falls er noch nicht aktiviert ist.
Siehe auch: MakeCrewMember, SelectCrew, SetCrewEnabled
