CreateArray

Kategorie: Script / Strings
Ab Engineversion: 4.9.6.0 CR

Beschreibung

Erzeugt ein neues, mit Nullen gefülltes Feld.

Syntax

array CreateArray (int Length);

Parameter

Length:
Länge des neuen Felds
Siehe auch: GetLength, SetLength
