GetCrewExtraData

Kategorie: Spieler
Ab Engineversion: 4.9.6.0 CR

Beschreibung

Dieser Befehl fragt im Clonk in der Spielerdatei gespeicherten Extra-Daten ab, die per SetCrewExtraData gesetzt wurden.

Syntax

any GetCrewExtraData (object pCrew, string strDataName);

Parameter

pCrew:
Crewobjekt, aus dessen Infosektion zusätzliche Daten abgefragt werden. 0 für lokalen Aufruf.
strDataName:
Name, dessen zugeordnete Daten abgefragt werden.

Anmerkung

Für Details siehe SetCrewExtraData.
Siehe auch: GetPlrExtraData, SetCrewExtraData
