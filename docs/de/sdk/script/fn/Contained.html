Contained

Kategorie: Objekte / Inhalt
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Liefert das Objekt, in dem sich pObj befindet. Rückgabewert Null, wenn pObj sich im Freien befindet.

Syntax

object Contained (object pObj);

Parameter

pObj:
Objekt, für welches die Überprüfung durchgeführt werden soll; Null für lokale Aufrufe
