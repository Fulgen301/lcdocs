SetSkyAdjust

Kategorie: Himmel
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Setzt die Himmelsmodulation. Die Himmelsfarben werden mit der angegebenen Farbe so verrechnet wie auch ColorByOwner-Grafiken.

Syntax

void SetSkyAdjust (int dwClrMod, int dwBackClr);

Parameter

dwClrMod:
Primäre Farbmodulation
dwBackClr:
Hintergrundfarbe. Wenn in dwClrMod das Alphabyte nicht 0 ist, wird diese Farbe als Hintergrund hinter den Himmel gelegt. So lassen sich auch hellere oder entsättigte Himmel realisieren.

Beispiel

SetSkyAdjust(RGBa(255,255,255,127), RGB(127,127,127));
Graut den Himmel aus.
Siehe auch: GetSkyAdjust, RGB, RGBa, SetGamma, SetMatAdjust, SetSkyFade
