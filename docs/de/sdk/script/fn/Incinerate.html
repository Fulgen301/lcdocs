Incinerate

Kategorie: Objekte
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Zündet das übergebene Objekt an. In dem jeweiligen Objekt wird ggf. Incineration() aufgerufen

Syntax

bool Incinerate (object pObj);

Parameter

pObj:
Objekt, welches entzündet werden soll.
Siehe auch: Extinguish, OCF_OnFire, OnFire
