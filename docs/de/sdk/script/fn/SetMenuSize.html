SetMenuSize

Kategorie: Objekte / Menü
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Ändert die Größe eines Objektmenüs, so dass die vorgegebene Anzahl an Menüeinträgen in Breite und Höhe hineinpassen.

Syntax

bool SetMenuSize (int iCols, int iRows, object pMenuObj);

Parameter

iCols:
0-50: Anzahl der Menüeinträge, die das Menü in der Breite anzeigen können soll
iRows:
0-50: Anzahl der Menüeinträge, die das Menü in der Höhe anzeigen können soll
pMenuObj:
[opt] Menüobjekt für das die Menügröße geändert werden soll. 0 wenn sich das Menu im aufrufenden Objekt befindet.
Siehe auch: AddMenuItem, CreateMenu, GetMenu, GetMenuSelection, SetMenuDecoration, SetMenuTextProgress
