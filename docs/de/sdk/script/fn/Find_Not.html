Find_Not

Kategorie: Objekte / Suche
Ab Engineversion: 4.9.6.0 CR

Beschreibung

Suchkriterium: findet genau alle Objekte, die nicht das angegebene Kriterium erfüllen.

Syntax

array Find_Not (array Criteria);

Parameter

Criteria:
Suchkriterium in Form einer Find_*-Funktion

Anmerkung

Für weitere Informationen und Beispiele zur Benutzung dieser Funktion sehe FindObjects.

Beispiel

FindObjects(Find_Distance(50), Find_Not(Find_Distance(30))
Ergibt eine Auflistung aller Objekte, die höchstens 50 - aber mehr als 30 - Pixel entfernt sind.
Siehe auch: FindObjects
