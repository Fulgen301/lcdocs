GameCallEx

Kategorie: Script / Funktionsaufruf
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Ruft szFunction im Szenario-Script und in Spielzielen, Spielregeln und Umweltobjekten auf. Sollte die Funktion nicht vorhanden sein, gibt die Funktion Null zurück und verursacht keine Fehlermeldung.

Syntax

any GameCallEx (string szFunction, any ...);

Parameter

szFunction:
Name der Funktion im Script
...:
[opt] Bis zu neun Parameter, die an die Funktion Script übergeben werden

Beispiel

GameCallEx("InitializePlayer",0);
Lässt das Script nochmal die Initialisierung für den ersten Spieler ausführen.
Siehe auch: Call, DefinitionCall, GameCall, ObjectCall
