GetPlrMagic

Kategorie: Spieler
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Stellt das Magiewissen eines bestimmten Spielers fest.

Syntax

bool / id GetPlrMagic (int iPlr, id id, int iIndex);

Parameter

iPlr:
Spieler, dessen Magiewissen abgefragt wird.
id:
Bei angegebenen id gibt die Funktion TRUE zurück, wenn der iPlr betreffenden Zauber zur Verfügung hat. iIndex wird hierbei ignoriert.
iIndex:
Bei angegebenem iIndex liefert die Funktion den indizierten Zauberspruch, der dem Spieler zur Verfügung steht.

Beispiele

if (GetPlrMagic(0,MLGT))
	Message("Spieler %s kann Blitze zaubern",0,GetPlayerName(0));
Wenn Spieler 0 Blitze zaubern (MLGT) kann, wird eine entsprechende Botschaft ausgegeben.
var spell;

spell = GetPlrMagic(0,0,4);
Message("Der fünfte Zauberspruch des Spielers ist %i",0,spell);
Gibt die ID des 5. Zauberspruchs des ersten Spielers aus.
