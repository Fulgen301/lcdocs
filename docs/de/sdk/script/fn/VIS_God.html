VIS_God

Kategorie: Objekte / Sichtbarkeit
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Konstante für GetVisibility/SetVisibility. Das Objekt ist nur im allgemeinen Viewport (Sichtfenster im Entwicklermodus oder Spielfläche im Netzwerk ohne Spieler) sichtbar.

Syntax

int VIS_God ();

Anmerkung

Dieser Wert ist als Engine-Konstante definiert. Die Funktionsklammern können daher optional weggelassen werden.

Beispiel

SetVisibility(VIS_God(), FindObject(GIDL));
Eine Goldstatue ist nur noch im allgemeinen Viewport sichtbar.
Siehe auch: GetVisibility, SetVisibility, VIS_All, VIS_Allies, VIS_Enemies, VIS_LayerToggle, VIS_Local, VIS_None, VIS_OverlayOnly, VIS_Owner
