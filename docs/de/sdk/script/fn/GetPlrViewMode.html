GetPlrViewMode

Kategorie: Spieler / Sicht
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Fragt den aktuellen Sichtmodus eines Spielers ab. Die Funktion gibt einen der folgenden Werte zurück:
  • 0: Cursor - Die Sicht des Spielers folgt dem aktuell selektierten Clonk (siehe GetCursor)
  • 1: Target - Die Sicht folgt einem anderen Objekt, welches durch den Befehl SetPlrView im Script gesetzt wurde. Siehe auch GetPlrView.
  • 2: Scroll - Der Spieler bewegt seine Sicht unabhängig von der Position eines Objekts. Dies geschieht, wenn der Spieler die Maus an den Rand seines Sichtbereichs zieht.

Syntax

int GetPlrViewMode (int iPlr);

Parameter

iPlr:
Nummer des Spielers, dessen Sichtmodus abgefragt werden soll.

Anmerkung

Da die Maus nicht über Netzwerk synchronisiert wird, gibt die Funktion für Record, Replay und Netzwerkspiele immer -1 zurück.
Siehe auch: GetPlrView, SetPlrView
