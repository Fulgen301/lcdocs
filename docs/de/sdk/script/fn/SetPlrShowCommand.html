SetPlrShowCommand

Kategorie: Spieler
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Markiert für einen Spieler ein bestimmtes Kommando. Ist die Darstellung deaktiviert, so wird sie aktiviert. Das markierte Kommando wird dann (so verfügbar) blinkend dargestellt.

Spielerkommandonummern:
Nummer Bedeutung
1 Links
2 Rechts
3 Springen
4 Stopp
5 Werfen
6 Graben
7 Special
8 Special 2
Für Doppelkommandos (z.B. Doppel-Graben) muss jeweils 128 auf den Kommandowert aufaddiert werden.

Syntax

bool SetPlrShowCommand (int iPlr, int iCom);

Parameter

iPlr:
Spieler
iCom:
Zu markierendes Spielerkommando.

Anmerkung

Diese Funktion beeinflusst nicht die Darstellung des Steuerungsblocks (siehe SetPlrShowControl).
Siehe auch: SetPlrShowControl, SetPlrShowControlPos
