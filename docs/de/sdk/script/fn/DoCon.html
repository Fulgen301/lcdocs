DoCon

Kategorie: Objekte / Status
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Verändert die Fertigstellung eines Objekts. Die meisten Lebewesen und Bäume können dadurch vergrößert oder verkleinert werden. Gebäude können zu Baustellen zurückgesetzt und Baustellen vorwärts oder rückwärts gebaut werden.
Sinkt die Fertigstellung eines Objekts auf oder unter 0, wird dieses gelöscht.

Syntax

bool DoCon (int iChange, object pObj);

Parameter

iChange:
Veränderung der Fertigstellung in Prozentpunkten; positiv oder negativ.
pObj:
[opt] Objekt, dessen Fertigstellung geändert werden soll. 0 bei lokalem Aufruf.

Beispiele

var obj;
while (obj=FindObject(0,0,0,0,0,OCF_Construct(),0,0,NoContainer(),obj)) DoCon(100,obj);
Baut alle Baustellen fertig.
DoCon(-50, GetCursor(plr));
Raubt dem angewähltem Mannschaftsmitglied von Spieler plr die Hälfte seiner normalen Größe. Wenn es vorher ausgewachsen war, hat es danach halbe normale Größe, wenn es halbe normale Größe hatte ist es danach weg.
Siehe auch: GetCon, SetCon
