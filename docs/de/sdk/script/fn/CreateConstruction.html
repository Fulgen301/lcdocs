CreateConstruction

Kategorie: Objekte / Erzeugung
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Sollte statt CreateObject benutzt werden, wenn ein Gebäude oder eine Baustelle erzeugt werden soll. Bei lokalen Aufrufen dieser Funktion wird die angegebene Position zu Offset-Koordinaten zum aufrufenden Objekt. Wenn fCheckSite 1 ist, wird die Baustelle vorher auf ihre Eignung überprüft. Ist die Position ungeeignet (z.B. durch unebenen Untergrund oder andere Gebäude), wird beim aufrufenden Objekt eine entsprechende Meldung ausgegeben und die Funktion gibt 0 zurück.

Syntax

object CreateConstruction (id id, int iXOffset, int iYOffset, int iOwner, int iCompletion, bool fTerrain, bool fCheckSite);

Parameter

id:
Typ des zu erstellenden Objekts
iXOffset:
X-Koordinate
iYOffset:
Y-Koordinate
iOwner:
Besitzer des zu erstellenden Objekts (Index eines Spielers oder NO_OWNER für keinen Besitzer)
iCompletion:
Gibt die Fertigstellung des Objekts in Prozent an (muss mindestens 1 sein)
fTerrain:
Bestimmt, ob die Landschaft an die Baustelle angepasst werden soll (inkl. Fundament).
fCheckSite:
Überprüft die Baustelle vorher auf ihre Eignung

Beispiele

CreateConstruction( HUT2, 150,100,  0,100,1 );
Erschafft eine fertige Holzhütte bei den Koordinaten 150/100 für Spieler 1 und passt die Landschaft an das Gebäude an.
CreateConstruction( HUT3, 200,100, NO_OWNER, 30,1 );
Errichtet eine zu 30% fertiggestellte Holzhütte bei den Koordinaten 200/100 ohne Besitzer wobei hier die Landschaft ebenfalls angepasst wird.
