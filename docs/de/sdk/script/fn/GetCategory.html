GetCategory

Kategorie: Objekte / Kategorie
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Fragt die Objektkategorie eines Objekts ab. Die Objektkategorie wird normalerweise über die DefCore gesetzt, und danach höchstens in seltenen Fällen mit SetCategory geändert. Sie bestimmt das allgemeine Verhalten eines Objekts und dessen Platzierung. Eine mit GetCategory abgefragt Kategorie kann mit dem &-Operator gegen eine C4D_-Konstante geprüft werden.

Syntax

int GetCategory (object pObj, id idDef);

Parameter

pObj:
[opt] Objekt, dessen Kategorie abgefragt wird. 0 bei lokalem Aufruf.
idDef:
[opt] Wenn angegeben, wird die Kategorie dieser Definition zurückgegeben.

Beispiel

if (GetCategory(obj) & C4D_Structure()) Incinerate(obj);
Zündet obj an, wenn es ein Gebäude ist.
