GetRGBaValue

Kategorie: Arithmetik
Ab Engineversion: 4.9.5.0 CE

Beschreibung

Liefert einer der folgenden Farbwerte des angegebenen 32Bit-Farbwerts:
iSelect Rückgabewert
0 Alpha-Wert
1 Rotanteil
2 Grünanteil
3 Blauanteil

Syntax

int GetRGBaValue (int Val, int iSelect);

Parameter

Val:
32Bit-Farbwert
iSelect:
0-3: Auswahl des gewünschten Rückgabewerts

Beispiel

GetRGBaValue(GetColorDw(GetCursor()),2);
Liefert den Grünanteil des aktuell ausgewählten Clonks
Siehe auch: RGBa, SetRGBaValue
