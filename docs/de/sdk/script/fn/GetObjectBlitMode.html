GetObjectBlitMode

Kategorie: Objekte / Darstellung
Ab Engineversion: 4.9.5.0 CE (erweitert ab 4.9.6.0 CR)

Beschreibung

Liefert den aktuellen Zeichenmodus eines Objekts.

Syntax

int GetObjectBlitMode (object pObj, int iOverlayID);

Parameter

pObj:
[opt] Objekt, dessen Zeichenstil abgefragt werden soll. 0 bei lokalem Aufruf.
iOverlayID:
[opt] Wenn angegeben, wird stattdessen der Blitmodus dieses Grafikoverlays abgefragt. Das Overlay muss vorher mit SetGraphics erstellt worden sein. Ab CR.

Anmerkungen

Für eine Übersicht über die Zeichenmodi, siehe SetObjectBlitMode().
Für ein Beispiel siehe SetObjectBlitMode().
Siehe auch: GetClrModulation, SetObjectBlitMode
