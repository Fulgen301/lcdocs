GetActionTarget

Kategorie: Objekte / Aktivität
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Liefert die aktuellen Aktivitätszielobjekte eines Objekts.

Syntax

object GetActionTarget (int iTargetIndex, object pObj);

Parameter

iTargetIndex:
0 für pTarget oder 1 für pTarget2
pObj:
[opt] Objekt, 0 bei lokalem Aufruf.

Beispiel

ObjectCall( GetActionTarget( ), "ChopDown");
Im StartCall der Fällen-Aktivität eines Clonks ausgeführt, fällt es die meisten Bäume sofort.
Siehe auch: GetActTime, GetAction, GetPhase, ObjectSetAction, SetAction, SetActionTargets, SetPhase
