SetOwner

Kategorie: Objekte / Status
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Ändert den Besitzer eines Objekts.

Syntax

bool SetOwner (int iNewOwner, object pObj);

Parameter

iNewOwner:
Spielernummer des neuen Besitzers. -1 für keinen Besitzer.
pObj:
[opt] Objekt, dessen Besitzer geändert wird. 0 bei lokalem Aufruf.

Anmerkung

Dieser Aufruf ändert auch den Controller des Objektes auf den neuen Besitzer

Beispiel

protected func Entrance(pContainer) { SetOwner(GetOwner(pContainer)); return(); }
Das Objekt ändert seinen Besitzer, wenn es eingesammelt wird.
Siehe auch: GetOwner, MakeCrewMember, SetController
