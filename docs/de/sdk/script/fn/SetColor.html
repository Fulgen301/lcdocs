SetColor

Kategorie: Objekte / Darstellung
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Setzt die ColorByOwner-Färbung eines Objekts. Dieser Aufruf entspricht dem Setzen eines neuen Besitzers mit SetOwner bei einem Spieler mit der Spielerfarbe iColor.

Syntax

bool SetColor (int iColor, object pObj);

Parameter

iColor:
Neue Farbe für ColorByOwner-Flächen
pObj:
[opt] Zielobjekt. 0 bei lokalem Aufruf.

Beispiel

SetColor(2, GetCursor(0));
Färbt die nach dem Besitzer gefärbten Flächen des ausgewählten Clonk des ersten Spielers rot.
Siehe auch: GetColor, SetColorDw
