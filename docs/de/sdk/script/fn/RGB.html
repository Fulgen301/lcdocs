RGB

Kategorie: Arithmetik
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Setzt einen 32Bit-Farbwert aus den drei Grundfarben zusammen. 0-255 für alle Parameter.

Syntax

int RGB (int byR, int byG, int byB);

Parameter

byR:
Rotanteil für die Farbe
byG:
Grünanteil für die Farbe
byB:
Blauanteil für die Farbe

Beispiel

SetSkyAdjust(RGB(255,0,0));
Färbt den Himmel rot.
Siehe auch: HSL, HSLa, RGBa
