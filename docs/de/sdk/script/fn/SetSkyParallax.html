SetSkyParallax

Kategorie: Himmel
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Verändert Parallaxität und Scrollgeschwindigkeit des Himmels.

Syntax

void SetSkyParallax (int iMode, int iParX, int iParY, int iXDir, int iYDir, int iX, int iY);

Parameter

iMode:
Scrollmodus. 0 ist Standard; bei 1 bewegt sich der Himmel mit dem Wind.
iParX:
Parallaxität in X-Richtung. Bei 10 ist der Himmel fest mit der Landschaft, mit größeren Werten beginnt die Parallaxität. Bei 0 wird der vorherige Wert beibehalten.
iParY:
Parallaxität in Y-Richtung
iXDir:
Feste, horizontale Bewegungsgeschwindigkeit des Himmels
iYDir:
Feste, vertikale Bewegungsgeschwindigkeit des Himmels
iX:
X-Scrollposition des Himmels
iY:
Y-Scrollposition des Himmels

Anmerkung

Wird für einen der Parameter SkyPar_Keep() angegeben, wird der vorherige Wert beibehalten.

Beispiel

SetSkyParallax(1, 20,20, 0,0, SkyPar_Keep(),SkyPar_Keep());
Setzt den Himmel auf parallax und mit dem Wind bewegend.
