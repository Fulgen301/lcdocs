EffectCall

Kategorie: Effekte
Ab Engineversion: 4.9.5.0 CE

Beschreibung

Führt einen Funktionsaufruf im angegebenen Effekt durch. Der Aufruf erhält den Befehlskontext des Effektes, und die ersten beiden Aufrufparameter sind immer Zielobjekt und Effektindex. Die weiteren Parameter können frei gewählt werden.

Syntax

any EffectCall (object pTarget, int iNumber, string szCommand, any ...);

Parameter

pTarget:
[opt] Zielobjekt des Effektes, auf dessen Variablenspeicher zugegriffen wird. 0 für globale Effekte.
iNumber:
Effektnummer. Effektnummern werden an Effekt-Callbackfunktionen übergeben, sind Rückgabewert von AddEffect und lassen sich über GetEffect aus dem Effektnamen herausfinden.
szCommand:
Aufzurufender Befehl. Es wird automatisch Fx[Effektname] vorangestellt; dies sollte nicht mit angegeben werden.
...:
Zusätzliche Aufrufparameter

Anmerkung

Für Beispiele und weitere Hinweise siehe Effekte-Dokumentation.
Siehe auch: AddEffect, ChangeEffect, CheckEffect, EffectVar, Effekte-Dokumentation, GetEffect, GetEffectCount, RemoveEffect
