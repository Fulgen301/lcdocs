ShakeFree

Kategorie: Landschaft
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Löst innerhalb eines kreisförmigen Bereiches halbfestes Material in der Landschaft. Für gelöstes Material wird neu geprüft, ob es in darunterliegendes Material fallen kann. Dadurch kann man zum Beispiel Höhlendecken einstürzen lassen.

Syntax

void ShakeFree (int iX, int iY, int iRadius);

Parameter

iX:
X-Mittelposition des freigeschüttelten Kreises. Immer global.
iY:
Y-Mittelposition des freigeschüttelten Kreises. Immer global.
iRadius:
Radius des freigeschüttelten Kreises

Beispiel

ShakeFree(LandscapeWidth()/2, LandscapeHeight()/2, Distance(LandscapeWidth(), LandscapeHeight())/2);
Schüttelt die gesamte Landschaft frei. Achtung: Extrem rechenintensiv :)
Siehe auch: CastPXS, CastPXSX, ShakeObjects
