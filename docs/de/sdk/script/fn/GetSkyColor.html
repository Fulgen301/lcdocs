GetSkyColor

Kategorie: Himmel
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Liefert den Farbwert des Himmels. Mit iRGB kann angegeben werden ob entweder Rot (iRGB ist 0), Grün (iRGB ist 1) oder Blau (iRGB ist 2) zurück gegeben werden soll. Mit iIndex wird festgelegt von welchem Palettenindex (0 - 19) der Wert zurück gegeben werden soll.

Syntax

int GetSkyColor (int iIndex, int iRGB);

Parameter

iIndex:
Farb-Index in der Clonk-Palette.
iRGB:
Rückgabetyp bestimmen.
Siehe auch: SetSkyColor, SetSkyFade
Cucaracha, Mai 2002
