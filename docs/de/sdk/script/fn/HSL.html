HSL

Kategorie: Arithmetik
Ab Engineversion: 4.9.5.0 CE

Beschreibung

Setzt einen 32Bit-Farbwert aus den drei Parametern zusammen. 0-255 für alle Parameter.

Syntax

int HSL (int byH, int byS, int byL);

Parameter

byH:
Farbton der Farbe. 0 ist Rot, 85 ist Grün, 170 Blau und 255 wieder Rot.
byS:
Sättigung der Farbe. Dies ist der Farbkontrast.
byL:
Helligkeit der Farbe. Helligkeit gleich null entspräche RGB(0,0,0)

Beispiele

SetSkyAdjust(HSL(0,255,127));
Färbt den Himmel rot.
SetColorDw(HSL(Random(256),255,127),GetHiRank(0));
Färbt den höchstrangigsten Clonk des ersten Spielers in einem zufälligen kräftigem Farbton.
Siehe auch: HSLa, RGB, RGBa
