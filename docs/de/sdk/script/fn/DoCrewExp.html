DoCrewExp

Kategorie: Objekte / Mannschaft
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Gibt einem Clonk Erfahrungspunkte.

Syntax

bool DoCrewExp (int iChange, object pObj);

Parameter

iChange:
Veränderung des Erfahrungswertes
pObj:
Crewobjekt, dem Erfahrung gegeben werden soll

Anmerkung

Der Clonk muss eine Infosektion haben, das heißt er muss ein CrewMember sein und einem Spieler gehören.

Beispiel

DoCrewExp(100, GetCursor());
Sound("Applause", 1); GameOver();
Gibt dem ausgewählten Clonk des ersten Spielers 100 Punkte Erfahrung, spielt einen Applaus ab und beendet die Runde.
Siehe auch: GetRank
