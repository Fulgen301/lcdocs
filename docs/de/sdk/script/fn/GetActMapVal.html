GetActMapVal

Kategorie: Get*Val
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Fragt Aktionsdaten aus der ActMap.txt anhand deren dortigem Namen ab.

Syntax

any GetActMapVal (string strEntry, string strAction, id idDef, int iEntryNr);

Parameter

strEntry:
Name des abzufragenden Eintrags
strAction:
[opt] Name der Aktion, von der der Eintrag abgefragt werden soll
idDef:
[opt] Objektdefinition, von der die Aktionsdaten abgefragt werden sollen. Bei Übergabe von 0 wird die Definition des aufrufenden Objekts angenommen.
iEntryNr:
[opt] Nummer des abzufragenden Wertes, wenn der Eintrag mehrere Werte enthält.

Beispiel

var iActLength = GetActMapVal("Length", GetAction()) * GetActMapVal("Delay", GetAction());
Berechnet die Länge in Ticks, die die aktuelle Action dauert
Siehe auch: GetDefCoreVal, GetObjectInfoCoreVal, GetObjectVal, GetPlayerInfoCoreVal, GetPlayerVal, GetScenarioVal
