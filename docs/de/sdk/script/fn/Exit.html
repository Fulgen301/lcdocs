Exit

Kategorie: Objekte / Position
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Versetzt ein Objekt aus seinem Behälter heraus, und setzt optional seine Geschwindigkeit.

Syntax

bool Exit (object pObj, int iX, int iY, int iR, int iXDir, int iYDir, int iRDir);

Parameter

pObj:
[opt] Nach draußen zu bewegendes Objekt. 0 bei lokalem Aufruf.
iX:
[opt] X-Austrittsposition. Offset zum aufrufenden Objekt bei lokalem Aufruf.
iY:
[opt] Y-Austrittsposition. Offset zum aufrufenden Objekt bei lokalem Aufruf.
iR:
[opt] Rotation des Austrittsobjekts in Grad.
iXDir:
[opt] X-Geschwindigkeit des Austrittsobjekts. Es gilt der übergebene Wert ohne Genauigkeitsteilung (Precision 1).
iYDir:
[opt] Y-Geschwindigkeit des Austrittsobjekts. Es gilt der übergebene Wert ohne Genauigkeitsteilung (Precision 1).
iRDir:
[opt] Rotationsgeschwindigkeit des Austrittsobjekts.

Beispiel

Exit(Contents(), 0,-20, Random(360), 0,-15, Random(11)-5);
Bewegt das erste Inhaltsobjekt oberhalb aus dem aufrufenden Objekt heraus, dreht es zufällig und schleudert es nach oben.
