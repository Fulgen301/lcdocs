GetNeededMatStr

Kategorie: Objekte / Status
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Liefert die noch benötigten Baumaterialien des Objekts

Syntax

string GetNeededMatStr (object pObj);

Parameter

pObj:
Objekt, dessen Baumaterialsbedarf abgefragt wird. 0 bei lokalem Aufruf.

Beispiel

/* Wird von der Bau-Action des Clonks aufgerufen */
private func Building()
{
  /* benötigtes Baumaterial abfragen */
  var string=GetNeededMatStr(GetActionTarget());
  /* Ausgeben */
  Message(Format("%s",string),this());
  return(1);
}
