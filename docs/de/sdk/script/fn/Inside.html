Inside

Kategorie: Arithmetik
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Gibt TRUE zurück, wenn iVal >= iRange1 und iVal <= iRange2 ist.

Syntax

bool Inside (int iVal, int iRange1, int iRange2);

Parameter

iVal:
Eingabewert
iRange1:
Untergrenze für den Wert
iRange2:
Obergrenze für den Wert
Siehe auch: BoundBy
