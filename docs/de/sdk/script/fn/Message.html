Message

Kategorie: Nachrichten
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Zeigt eine Nachricht an. Diese kann nach dem C-Formatierungsstil formatiert werden (siehe Format)

Syntax

bool Message (string strMessage, object pObj,  ...);

Parameter

strMessage:
Nachricht, die angezeigt werden soll.
pObj:
[opt] Objekt, über dem die Nachricht angezeigt werden soll. Wenn nicht angegeben oder 0 wird die Nachricht global angezeigt.
...:
[opt] Zusätzliche Formatierungsparameter

Anmerkung

Durch Verwendung des Zeichens "|" kann ein Zeilenvorschub erreicht werden. "<c AARRGGBB>Test</c>" gibt "Test" farbig aus (AA: Transparenz, RR: Rot-, GG: Grün. BB: Blauanteil; hexadezimale Angaben). {{C4ID}} zeigt ein Bild des Objektes mit der ID C4ID; durch Angabe von Zahlen können verschiedene Teile der Grafik verwendet werden, z.B. {{C4ID:2}}.

Beispiel

Message("Hallo %s!|Ich bin dein Clonk %s!", this(), GetPlayerName(GetOwner()), GetName());
Wird dieser Script in einem Clonk ausgeführt, so erscheint über seinem Kopf der entsprechende Text (z.B. "Hallo Twonky! Ich bin dein Clonk Yoki!")
Siehe auch: AddMessage, CustomMessage, Log, PlayerMessage, PlrMessage
