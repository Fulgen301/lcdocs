GetDefinition

Kategorie: Objekte
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Liefert das ID der indizierten von der Engine in dieser Runde geladenen Objektdefinition der angegebenen Kategorie (dwCategory 0 für alle Objektkategorien).

Syntax

id GetDefinition (int iIndex, int dwCategory);

Parameter

iIndex:
Index der gesuchten Definition.
dwCategory:
[opt] Die Kategory, in der die gesuchte Definition ist.

Beispiel

siehe C4D_Goal.
Siehe auch: C4D_Goal, C4D_Knowledge, C4D_Living, C4D_Object, C4D_Rule, C4D_StaticBack, C4D_Structure, C4D_Vehicle, GetHomebaseMaterial, GetPlrKnowledge
