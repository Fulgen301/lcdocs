SetContactDensity

Kategorie: Objekte / Status
Ab Engineversion: 4.9.6.0 CR

Beschreibung

Legt fest, welche Dichte ein Material haben muss, damit das Objekt mit diesem kollidiert. Standardmäßig liegt dieser Wert bei 50 (dichtes Material).

Syntax

bool SetContactDensity (int iDensity, object pObj);

Parameter

iDensity:
Dichte des Materials, mit dem das Objekt kollidieren soll.
pObj:
[opt] Zielobjekt. 0 bei lokalem Aufruf.

Beispiel

SetContactDensity(25, GetCursor());
Ändert die Kontaktdichte des ausgewähten Clonks des ersten Spielers so, dass dieser auch auf Flüssigkeiten laufen kann.
