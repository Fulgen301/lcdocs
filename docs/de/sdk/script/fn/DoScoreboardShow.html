DoScoreboardShow

Kategorie: Interface
Ab Engineversion: 4.9.6.0 CR

Beschreibung

Ändert den Sichtbarkeitszähler für das Scoreboard. Der intern verwaltete Zähler beginnt bei 0. Ist er größer als 0, wird das Scoreboard gezeigt. Ist er kleiner als 0, wird es verborgen. Ist er gleich 0, wird das Scoreboard nur dann gezeigt, wenn der Spieler die entsprechende Taste (normalweise Tab) drückt.

Syntax

bool DoScoreboardShow (int iChange, int iForPlr);

Parameter

iChange:
Änderung der Sichtbarkeit des Scoreboards.
iForPlr:
[opt] Wenn größer 0, Spielernummer+1 des Spielers, für den die Scoreboard-Sichtbarkeit geändert werden soll.

Beispiel

Für ein ausführliches Beispiel siehe SetScoreboardData.
Siehe auch: ScoreboardCol, SetScoreboardData, SortScoreboard
