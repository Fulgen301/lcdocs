ScoreboardCol

Kategorie: Interface
Ab Engineversion: 4.9.6.0 CR

Beschreibung

Wandelt eine Objektdefinitions-ID in eine Zahl um, die als Spaltenidentifikator für SetScoreboardData verwendet werden kann. Diese Funktion ist dann interessant, wenn ein Objekt auf das Scoreboard zugreifen will, aber dabei sicher stellen muss, dass es nicht dieselbe ID wie ein anderes Objekt verwendet.

Syntax

int ScoreboardCol (id idDef);

Parameter

idDef:
C4ID, die in einen Scoreboard-Spaltenindex umgewandelt werden soll. Für Scoreboards auf Objektbasis sollte hier die ID der Objektdefinition genommen werden, die die Scoreboardfunktionen bereit stellt.
Siehe auch: DoScoreboardShow, SetScoreboardData, SortScoreboard
