ReloadParticle

Kategorie: Entwickler
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Aktualisiert eine Partikeldefinition. Dieser Befehl kann im Entwicklermodus in der Konsole eingegeben werden, nachdem die Definition auf der Festplatte verändert wurde. Sie muss zuvor geladen gewesen sein.

Syntax

bool ReloadParticle (string szParticle);

Parameter

szParticle:
Name des neu zu ladenden Partikeltyps.

Beispiel

ReloadParticle("Smoke");
Das Rauchpartikel wird neu geladen.
