Find_Category

Kategorie: Objekte / Suche
Ab Engineversion: 4.9.6.0 CR

Beschreibung

Suchkriterium: sucht alle Objekte mit der entsprechenden Kategorie.

Syntax

array Find_Category (int iCategory);

Parameter

iCategory:
Objektkategorie der zu suchenden Objekte

Anmerkung

Für weitere Informationen und Beispiele zur Benutzung dieser Funktion sehe FindObjects.

Beispiele

FindObjects(Find_Category(C4D_Structure))
Gibt eine Liste aller im Spiel befindlichen Gebäude zurück.
FindObjects(Find_Category(C4D_Rule | C4D_Goal))
Liefert alle aktiven Spielregeln und Spielziele zurück. Man beachte, dass sich die Kategorien mit "|" kombinieren lassen. In diesem Fall trifft das Kriterium auf alle Objekte zu, bei denen mindestens eine gesetzte Kategorie übereinstimmt.
Siehe auch: FindObjects
