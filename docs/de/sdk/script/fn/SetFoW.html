SetFoW

Kategorie: Spieler / Sicht
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Schaltet den Nebel des Krieges für einen Spieler ein oder aus.

Syntax

bool SetFoW (bool fEnabled, int iPlayer);

Parameter

fEnabled:
Wenn TRUE, wird der Kriegsnebel aktiviert, bei FALSE deaktiviert.
iPlayer:
Nummer des Spielers.

Beispiel

var i = 0;
for (var plrnum = GetPlayerCount (); plrnum; i++)
  if (GetPlayerName (i)) {
    plrnum--;
    SetFoW (0, i);
  }
Schaltet den Kriegsnebel für alle Spieler aus.
Siehe auch: SetPlrView, SetPlrViewRange
