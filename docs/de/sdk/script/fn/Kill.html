Kill

Kategorie: Objekte / Lebewesen
Ab Engineversion: 4.9.1.0 GWE (erweitert ab 4.9.5.0 CE)

Beschreibung

Tötet ein Objekt. Nur Lebewesen.

Syntax

bool Kill (object pObj, bool fForced);

Parameter

pObj:
[opt] Objekt, das getötet wird. 0 bei lokalem Aufruf.
fForced:
[opt] Wenn gesetzt, wird das Objekt auch dann getötet, wenn ein Effekt das zu unterbrechen versucht. Ab CE.

Beispiel

Kill(FindObject(0, 0,0,0,0, OCF_Alive()));
Tötet ein Lebewesen.
Siehe auch: Punch, RemoveObject
