Log

Kategorie: Nachrichten
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Gibt einen Text im Messageboard am unteren Bildschirmrand aus. Die Parameter können über den C-Formatierungsstil in den Text eingebunden werden. Siehe dazu auch Format.

Syntax

void Log (string szMessage,  ...);

Parameter

szMessage:
Auszugebender Text
...:
[opt] Zusätzliche Parameter, mit denen der Text formatiert wird

Beispiel

Log("Moin, %s.", GetPlayerName(0));
Begrüßt den Spieler
Siehe auch: DebugLog, Format, Message
