SelectMenuItem

Kategorie: Objekte / Menü
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Setzt die Auswahl in einem Menu.

Syntax

bool SelectMenuItem (int iItem, object pMenuObj);

Parameter

iItem:
Nummer des MenuItems, das selektiert werden soll. Der erste Eintrag hat die Nummer 0.
pMenuObj:
[opt] Menüobjekt für das die Selektion geändert werden soll. 0 wenn sich das Menu im aufrufenden Objekt befindet.

Beispiel

SelectMenuItem kann zusammen mit GetMenuSelection dazu verwendet werden, ein Menu zu aktualisieren, ohne dabei die Selektion zu verändern:
var iSelection = -1;
// Menu bereits geöffnet?
if(GetMenu())
  iSelection = GetMenuSelection();

// [ Hier Menu (neu) erstellen... ]

// Selektion wiederherstellen?
if(iSelection >= 0)
  SelectMenuItem(iSelection);
