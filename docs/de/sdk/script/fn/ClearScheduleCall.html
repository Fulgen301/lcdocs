ClearScheduleCall

Kategorie: Script
Ab Engineversion: 4.9.5.0 CE

Beschreibung

Bricht durch ScheduleCall erzeugte verzögerte Funktionsaufrufe ab. Alle momentan wartenden Funktionsaufrufe an die angegebene Funktion und das angegebene Objekt werden abgebrochen.

Syntax

bool ClearScheduleCall (object pObj, string szFunction);

Parameter

pObj:
Objekt, für das verzögerte Funktionsaufrufe abgebrochen werden sollen. 0 für aufrufendes Objekt.
szFunction:
Name der Funktion, die durch die abzubrechenden Aufrufe aufgerufen würde.

Beispiel

ClearScheduleCall(FindObject(CLNK), "Redefine");
Bricht alle verzögerten "Redefine"-Aufrufe an den ersten Clonk ab.
