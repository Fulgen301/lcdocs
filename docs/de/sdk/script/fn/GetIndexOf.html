GetIndexOf

Kategorie: Variablen
Ab Engineversion: 4.9.7.0 CR

Beschreibung

Gibt den Index von value im Array aArray zurück. Es wird -1 zurückgegeben, wenn value nicht in aArray enthalten ist.

Syntax

int GetIndexOf (any value, array aArray);

Parameter

value:
Wert dessen Typ abgefragt werden soll.
aArray:
Array, welches durchsucht werden soll.
