_inherited

Kategorie: Script
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Ruft die überladene Funktion auf. Funktionen überladen sich, wenn sie beide den gleichen Namen haben. Die Funktion ist im Gegensatz zu inherited außerdem failsafe. Das heißt, dass es zu keinem Fehler führt, wenn die Ursprungsfunktion gar nicht existiert.

Syntax

any _inherited (any ...);

Parameter

...:
Aufrufparameter

Beispiel

#strict

#appendto CLNK

Initialize:
  CreateContents(FLNT);
  return(_inherited(Par(), Par(1)));
Gibt jedem Clonk zu Beginn einen Feuerstein.
Siehe auch: inherited
