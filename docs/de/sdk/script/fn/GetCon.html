GetCon

Kategorie: Objekte / Status
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Liefert die Fertigstellung eines Objekts in Prozent. Bei Baustellen ist dies die Fertigstellung, während es bei Tieren und Pflanzen die Größe angibt (Siehe StretchGrowth/OverSize in der DefCore)

Syntax

int GetCon (object pObj);

Parameter

pObj:
[opt] Objekt dessen Fertigstellung abgefragt werden soll.

Beispiel

if (GetCon(GetCursor()) > 50) DoCon(-10, GetCursor());
Schrumpft den ausgewählten Clonk des ersten Spielers um 10% seiner Normalgröße, wenn er nicht schon auf halbe Größe geschrumpft ist.
Siehe auch: DoCon, OCF_Fullcon, SetCon
