HSLa

Kategorie: Arithmetik
Ab Engineversion: 4.9.5.0 CE

Beschreibung

Setzt einen 32Bit-Farbwert aus den drei Parametern und einem Durchsichtigkeitswert zusammen. 0-255 für alle Parameter.

Syntax

int HSLa (int byH, int byS, int byL, int byA);

Parameter

byH:
Farbton der Farbe. 0 ist Rot, 85 ist Grün, 170 Blau und 255 wieder Rot.
byS:
Sättigung der Farbe. Dies ist der Farbkontrast.
byL:
Helligkeit der Farbe. Helligkeit gleich null entspräche RGB(0,0,0)
byA:
Durchsichtigkeit der Farbe.

Beispiel

SetClrModulation(HSLa(0,0,255,64),FindObject(TRE1));
Färbt einen zufälligen Baum absolut weiß und ein wenig durchsichtig.
Siehe auch: HSLa, RGB, RGBa
