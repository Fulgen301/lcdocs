GetMaterialVal

Kategorie: Get*Val
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Gibt einen Eigenschaftswert einer Materialdefinition nach seinem Namen in der Materialdatei zurück.

Syntax

any GetMaterialVal (string strEntry, string strSection, int iMaterial, int iEntryNr);

Parameter

strEntry:
Name des abzufragenden Eintrags
strSection:
[opt] Sektion des abzufragenden Eintrags. Wird 0 übergeben, werden alle Sektionen nach einem Eintrag mit dem entspr. Namen durchsucht.
iMaterial:
[opt] Nummer des Materials, von dem der Eintrag abgefragt werden soll. Bei Übergabe von 0 wird die Definition des aufrufenden Objekts angenommen.
iEntryNr:
[opt] Nummer des abzufragenden Wertes, wenn der Eintrag mehrere Werte enthält.

Beispiel

GetMaterialVal("BlastFree", "Material", GetMaterial(20, 0))
Überprüft, ob das Material rechts vom aufrufenden Objekt sprengbar ist.
Siehe auch: GetMaterial, Material
