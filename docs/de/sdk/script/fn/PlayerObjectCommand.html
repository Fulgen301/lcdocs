PlayerObjectCommand

Kategorie: Objekte / Commands
Ab Engineversion: 4.9.6.0 CR

Beschreibung

Führt das Kommando szCommand für den gerade ausgewählten Clonk von Spieler iPlr aus. Siehe SetCommand für mögliche Befehle.

Syntax

bool PlayerObjectCommand (int iPlr, string szCommand, object pTarget, int iX, int iY, object pTarget2, int iInterval, any Data, int iRetries, int iBaseMode);

Parameter

iPlr:
Spielernummer
szCommand:
Kommandoname (als String). Siehe untere Tabelle
pTarget:
[opt] Zielobjekt für Aktion
iX:
[opt] X-Zielkoordinate
iY:
[opt] Y-Zielkoordinate
pTarget2:
[opt] zweites Zielobjekt
iInterval:
[opt] Zeit (in Ticks/Frames), bis das Kommando abgebrochen wird
Data:
[opt] zusätzliche Daten zur Aktion
iRetries:
[opt] Anzahl der Wiederholungen (wenn das Kommando fehlschlägt), bis das Kommando abgebrochen wird.
iBaseMode:
[opt] Befehlsstapelmodus. Werte siehe AddCommand.

Beispiel

PlayerObjectCommand(0, "Throw");
Gibt dem ausgewählten Clonk des ersten Spielers den Befehl, das erste Inventarobjekt zu werfen.
Siehe auch: AddCommand, FinishCommand, GetCommand, SetCommand
