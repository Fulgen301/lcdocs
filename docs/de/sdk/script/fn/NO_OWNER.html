NO_OWNER

Kategorie: Spieler
Ab Engineversion: 4.9.5.0 CE

Beschreibung

Spielernummer: Kein Spieler. Diese Konstante kann als Besitzerangabe für diverse Funktionen wie CreateObject oder FindObjectOwner verwendet werden, um einen neutralen Spieler anzugeben.

Syntax

int NO_OWNER ();

Anmerkung

Dieser Wert ist als Engine-Konstante definiert. Die Funktionsklammern können daher optional weggelassen werden.

Beispiel

CreateObject(EFLN, 100,100, NO_OWNER);
Erzeugt einen besitzerlosen Flint.
Siehe auch: GetOwner, SetOwner
