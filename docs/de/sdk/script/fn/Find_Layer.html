Find_Layer

Kategorie: Objekte / Suche
Ab Engineversion: N/A

Beschreibung

Suchkriterium: sucht alle Objekte mit dem entsprechenden Layerobjekt.

Syntax

array Find_Layer (object pLayerObj);

Parameter

pLayerObj:
Layerobjekt in dem sich das Objekt befinden muss. nil für den globalen Layer.

Anmerkung

Für weitere Informationen und Beispiele zur Benutzung dieser Funktion sehe FindObjects.
Siehe auch: FindObjects, GetObjectLayer, SetObjectLayer
