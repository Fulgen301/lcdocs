SetNextMission

Kategorie: Global
Ab Engineversion: 4.9.7.0 CR

Beschreibung

Verändert den "Diese Runde weiterspielen"-Button beim Endbildschirm so, dass das angegebene Szenario beim Draufklicken gestartet wird. Wird szNextMissionFile auf das Szenario selbst gesetzt, kann so ein Neustart-Button erreicht werden.

Syntax

void SetNextMission (string szNextMissionFile, string szButtonText, string szTooltip);

Parameter

szNextMissionFile:
Name der Datei, welche gestartet werden soll.
szButtonText:
Text des Buttons
szTooltip:
Tooltip, welcher beim Überfahren des Buttons mit der Maus angezeigt werden soll.
