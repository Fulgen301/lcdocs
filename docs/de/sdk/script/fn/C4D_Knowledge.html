C4D_Knowledge

Kategorie: Objekte / Kategorie
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Objektkategorie: Bauplan.

Syntax

int  C4D_Knowledge ();

Anmerkung

Dieser Wert ist als Engine-Konstante definiert. Die Funktionsklammern können daher optional weggelassen werden.

Beispiel

if( GetCategory( 0, GetHomebaseMaterial( )) & C4D_Knowledge()) Message("Spieler 1 kann etwas kaufen, was er auch erforschen oder bauen könnte.");
Mit einem bitweisen And kann man überprüfen, ob zwei Kategorien in mindestens einer Kategorie übereinstimmen.
Siehe auch: GetCategory, GetDefinition, GetHomebaseMaterial, GetPlrKnowledge, SetCategory
