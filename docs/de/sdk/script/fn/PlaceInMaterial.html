PlaceInMaterial

Kategorie: Objekte / Erzeugung
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Erzeugt ein Objekt des gewünschten Typs irgendwo in der Landscape im gewünschten Material und liefert es zurück.

Syntax

object PlaceInMaterial (id idDef, int iMaterial, int iRetries);

Parameter

idDef:
Objekttyp
iMaterial:
Material
iRetries:
[opt] Anzahl Wiederholungen

Beispiele

PlaceInMaterial(FLNT, Material("Earth"));
Versucht einen Feuerstein in Erde zu erzeugen
var obj = PlaceInMaterial(FLAG, Material("Gold"));
if (obj)  SetOwner(0, obj);
if (!obj) Log("Konnte keine Flagge in Gold erzeugen!");
Versucht eine Flagge in Gold zu erzeugen und sie in den Besitz von Spieler 0 zu übergeben. Gelingt es nicht die Flagge zu erzeugen, wird eine Fehlermeldung geloggt
Siehe auch: CreateObject, PlaceObjects
