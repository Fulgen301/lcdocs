SetPlrViewRange

Kategorie: Spieler / Sicht
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Setzt die Sichtweite eines Objekts.

Syntax

bool SetPlrViewRange (int iRange, object pObj, bool fExact);

Parameter

iRange:
Sichtweite in Pixel.
pObj:
Objekt, dessen Sichtweite gesetzt wird. 0 bei lokalem Aufruf.
fExact:
Muss true sein, um Werte kleiner 128 wählen zu können.

Anmerkung

Aufgrund der Ungenauigkeit des Kriegsnebels wird die Sichtweite nur ungenaue Auswirkungen haben.

Beispiel

/* Wird bei Doppelklick auf Graben aufgerufen */
protected func Activate(pClonk)
{
	// Den Clonk blenden
	SetPlrViewRange(0, pClonk);
	// Schmerzensschrei 
	Sound("Scream", pClonk);
	return(1);
}
Mit Augentropfen verwechseltes Säureglas.
