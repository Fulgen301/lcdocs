ResetGamma

Kategorie: Global
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Setzt eine Gammarampe auf den Normalwert zurück. Dies entspricht SetGamma(RGB(0,0,0), RGB(128,128,128), RGB(255,255,255), iRampIndex);. Für weitere Informationen siehe SetGamma.

Syntax

void ResetGamma (int iRampIndex);

Parameter

iRampIndex:
0-7: Index der Gammarampe, die zurückgesetzt werden soll.

Beispiel

for (var i; i<8; i++) ResetGamma(i);
Deaktiviert alle Gammarampen.
Siehe auch: SetGamma
