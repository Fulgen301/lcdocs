GetPlrColorDw

Kategorie: Spieler
Ab Engineversion: 4.9.5.0 CE

Beschreibung

Fragt die Spielerfarbe ab. Die Spielerfarbe wird verwendet, um einige Besitzobjekte wie Clonks oder Flaggen, sowie Textnachrichten dieses Spielers einzufärben.

Syntax

int GetPlrColorDw (int iPlr);

Parameter

iPlr:
Abgefragter Spieler

Beispiel

Message("<c %x>%s</c> hat %d Punkte.", 0, GetPlrColorDw(0), GetPlayerName(0), GetScore(0));
Gibt Statusinformationen über den ersten Spieler aus, wobei dessen Spielername in seiner Spielerfarbe gefärbt wird.
Siehe auch: GetColorDw, SetColorDw
