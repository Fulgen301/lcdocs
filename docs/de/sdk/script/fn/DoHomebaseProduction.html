DoHomebaseProduction

Kategorie: Spieler
Ab Engineversion: 4.9.8.5 CR

Beschreibung

Verändert die Nachlieferung von kaufbaren Objekten in der Heimatbasis.

Syntax

bool DoHomebaseProduction (int iPlr, id idID, int iChange);

Parameter

iPlr:
Spielernummer des Spielers, dessen Heimatbasismaterial-Nachlieferung verändert werden soll
idID:
ID des Objekttyps, der schneller oder langsamer nachgeliefert werden soll.
iChange:
Positive oder negative Veränderung der nachgelieferten Objektmenge

Beispiel

DoHomebaseProduction(0, FLNT, -GetHomebaseProduction(0, FLNT));
Feuersteine werden für den ersten Spieler nicht mehr nachgeliefert.
Siehe auch: Buy, DoHomebaseMaterial, GetHomebaseMaterial, GetHomebaseProduction, Sell
