Sqrt

Kategorie: Arithmetik
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Zieht die Wurzel aus dem ersten Parameter. Ist der Parameter kleiner als 0, wird einfach 0 zurückgegeben.

Syntax

int Sqrt (int iValue);

Parameter

iValue:
Wert, aus dem die Wurzel gezogen werden soll

Beispiel

global func ObjectDistance(pObj1, pObj2)
{
  var dx = GetX(pObj1) - GetX(pObj2);
  var dy = GetY(pObj1) - GetY(pObj2);
  return(Sqrt(dx*dx+dy*dy));
}
Berechnet den Abstand zwischen zwei Objekten nach dem Satz des Pythagoras.
Anmerkung: ObjectDistance existiert bereits als Engine-Funktion.
Siehe auch: Distance, ObjectDistance
