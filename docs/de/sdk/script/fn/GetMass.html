GetMass

Kategorie: Objekte / Status
Ab Engineversion: 4.6.5.0 CP (erweitert ab 4.9.1.0 GWE)

Beschreibung

Liefert die Masse eines Objekts inklusive aller Inhaltsobjekte. Bei angegebenem idDef wird pObj ignoriert und stattdessen die Standardmasse der Objektdefinition zurückgegeben. Sind idDef und pObj 0, gilt das aufrufende Objekt als pObj.

Syntax

int GetMass (object pObj, id idDef);

Parameter

pObj:
[opt] Gewogenes Objekt. 0 bei lokalem Aufruf oder angegebenem idDef.
idDef:
[opt] Objektdefinition, von der das Standardgewicht ermittelt werden soll. 0 bei angegebenem pObj

Beispiel

/* Wird bei klick auf Spezial aufgerufen */
protected func ControlSpecial()
{
	// Magersüchtig?
	if(GetMass() < 50) // Normalmasse des CLNK: 50
		// Magensäure frisst sich nach aussen
		CastPXS("Acid", 150, 30);
return(1);
}
