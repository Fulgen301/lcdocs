Split2Components

Kategorie: Objekte / Existenz
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Zerlegt ein Objekt in seine Bestandteile. Die Bestandteile werden in zufälliger Drehung und mit geringer Geschwindigkeit in zufälliger Richtung an der Stelle erzeugt, an der sich das Objekt befand. Das Objekt wird anschließend vernichtet. Wenn sich das Objekt in einem Behälter befand, werden auch die Komponenten nach der Erschaffung in den Behälter versetzt.

Syntax

bool Split2Components (object pObj);

Parameter

pObj:
[opt] Objekt, das zerlegt werden soll. 0 bei lokalem Aufruf.

Anmerkung

Siehe Hinweis zu RemoveObject.

Beispiel

protected func ControlDigDouble()
  {
  var pTarget=Contents();
  if (!pTarget) return();
  Split2Components(pTarget);
  return(1);
  }
Script für einen Clonk: Auf Doppelgraben zerlegt er das erste Objekt, das er trägt.
Siehe auch: GetComponent, RemoveObject
