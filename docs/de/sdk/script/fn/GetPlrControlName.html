GetPlrControlName

Kategorie: Spieler
Ab Engineversion: 4.9.7.0 CR

Beschreibung

Gibt den Namen der Taste iCon von iPlr zurück.

Syntax

string GetPlrControlName (int iPlr, int iCon, bool fShort);

Parameter

iPlr:
Spieler, dessen Taste gesucht wird.
iCon:
Taste, die gesucht wird. Folgende Werte sind möglich:
Name Wert
CON_CursorLeft 0
CON_CursorToggle 1
CON_CursorRight 2
CON_Throw 3
CON_Up 4
CON_Dig 5
CON_Left 6
CON_Down 7
CON_Right 8
CON_Menu 9
CON_Special 10
CON_Special2 11
fShort:
Wenn true, wird die kurze Version der Taste zurückgegeben. (Beispiel: statt "< > |" wird "<" zurückgegeben.)

Beispiel

Log(GetPlrControlName(0, CON_Dig));
Gibt den Namen der Taste [Graben] vom ersten Spieler im Log aus. Dies ist "D", wenn dieser das unveränderte Tastaturprofil 1 hat.
