ClearMenuItems

Kategorie: Objekte / Menü
Ab Engineversion: 4.9.1.0 GWE

Beschreibung

Entfernt alle Einträge aus einem Menü.

Syntax

int ClearMenuItems (object pMenuObject);

Parameter

pMenuObject:
[opt] Objekt, aus dessen Menü alle Einträge geleert werden sollen. 0 bei lokalem Aufruf.

Beispiel

var pClonk=GetCursor();
CreateMenu(CLNK, pClonk, pClonk, 0, "", 0, 2);
AddMenuItem("", "", TIM1, pClonk,0,0,"Alter Text");

[...]

ClearMenuItems(pClonk);
AddMenuItem(0, "", TIM1, pClonk,0,0,"Neuer Text");
Öffnet zunächst ein Textfenster mit dem Inhalt "Alter Text" für den ausgewählten Clonk des ersten Spielers, und ändert die Beschriftung dann später auf "Neuer Text".
Siehe auch: AddMenuItem, CloseMenu, CreateMenu
