SetMenuDecoration

Kategorie: Objekte / Menü
Ab Engineversion: 4.9.6.0 CR

Beschreibung

Ändert die Dekoration eines Menüobjektes.

Syntax

bool SetMenuDecoration (id idNewDeco, object pMenuObj);

Parameter

idNewDeco:
ID des Objektes, dessen Grafik als Dekoration verwendet werden soll.
pMenuObj:
[opt] Menüobjekt, dessen Dekoration geändert werden soll. 0, wenn sich das Menu im aufrufenden Objekt befindet.
Siehe auch: AddMenuItem, CreateMenu, GetMenu, GetMenuSelection, SetMenuTextProgress
