SortScoreboard

Kategorie: Interface
Ab Engineversion: 4.9.6.0 CR

Beschreibung

Sortiert die Zeilen des Scoreboards nach den Extradaten einer Spalte. Wenn zwei Einträge in dieser Spalte die gleichen Werte besitzen, wird nicht umsortiert. So kann durch sukzessive Aufrufe dieser Funktion nach mehreren Spalten sortiert werden.

Syntax

bool SortScoreboard (int iColID, bool fReverse);

Parameter

iColID:
ID der Spalte, nach der sortiert werden soll.
fReverse:
[opt] Wenn wahr, wird in absteigender Reihenfolge sortiert. Ansonsten in aufsteigender Reihenfolge.

Beispiel

Für ein ausführliches Beispiel siehe SetScoreboardData.
Siehe auch: DoScoreboardShow, ScoreboardCol, SetScoreboardData
