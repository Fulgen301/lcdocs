Smoke

Kategorie: Umwelt / Effekte
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Erzeugt eine Rauchwolke an der angegebenen Position mit angegebener Farbe.

Syntax

void Smoke (int iX, int iY, int iLevel, int dwClr);

Parameter

iX:
X-Position der Rauchwolke. Offset bei lokalem Aufruf.
iY:
Y-Position der Rauchwolke. Offset bei lokalem Aufruf.
iLevel:
Größe des Rauchs
dwClr:
Farbe des Rauchs

Beispiel

Smoke(0,0,50, RGB(255,0,0));
Objektscript: Das Objekt stößt eine rote Rauchwolke aus.
Sven2, August 2002
