SetDir

Kategorie: Objekte / Aktivität
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Setzt die Richtung eines Objekts.

Syntax

int SetDir (int iDir, object pObj);

Parameter

iDir:
Aktionsrichtung. Vordefinierte Werte: DIR_Right() / DIR_Left()
pObj:
[opt] Objekt, dessen Richtung gesetzt werden soll. 0 bei lokalem Aufruf.

Anmerkungen

SetDir beeinflusst nur die grafische Darstellung das Objekts; für Veränderung der Bewegungsrichtung ist SetComDir zuständig!
Wenn bei der Aktion mehr als zwei Richtungen angegeben werden (siehe ActMap.txt), dürfen auch andere Werte als DIR_Right() und DIR_Left() übergeben werden.
Siehe auch: DIR_Left, DIR_Right, GetDir, SetAction
