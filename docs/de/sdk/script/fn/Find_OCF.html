Find_OCF

Kategorie: Objekte / Suche
Ab Engineversion: 4.9.6.0 CR

Beschreibung

Suchkriterium: Sucht alle Objekte, bei denen eine der angegebenen Object-Character-Flags gesetzt ist.

Syntax

array Find_OCF (int iOCF);

Parameter

iOCF:
Object-Character-Flag der zu suchenden Objekte

Anmerkung

Für weitere Informationen und Beispiele zur Benutzung dieser Funktion sehe FindObjects.

Beispiele

ObjectCount2(Find_OCF(OCF_InLiquid | OCF_InSolid))
Gibt die Anzahl der Objekte zurück, die sich in Flüssigkeit befinden oder im Boden stecken.
FindObject2(Find_OCF(OCF_Alive), Find_OCF(OCF_OnFire))
Gibt das erstbeste brennende Lebewesen zurück, falls ein solches existiert.
Siehe auch: FindObjects, Object character flags
