SetPosition

Kategorie: Objekte / Position
Ab Engineversion: 4.6.5.0 CP (erweitert ab 4.9.6.0 CR)

Beschreibung

Verschiebt ein Objekt an die angegebene Position. Dabei wird das Objekt an die exakte Pixelposition gesetzt, und eventuelle, interne Verschiebungen um Pixelbruchteile werden verworfen.

Syntax

int SetPosition (int iX, int iY, object pObj, bool fCheckBounds);

Parameter

iX:
Neue X-Position für das Objekt
iY:
Neue Y-Position für das Objekt
pObj:
[opt] Objekt dessen Position geändert werden soll. Null bei lokalem Aufruf.
fCheckBounds:
[opt] Nur wirksam für Objekte mit angegebenem BorderBound. Stellt sicher, dass die Position immer innerhalb der dem Objekt auferlegten Grenzen ist. (Ab CR)
Siehe auch: GetX, GetY
