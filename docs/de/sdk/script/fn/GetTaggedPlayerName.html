GetTaggedPlayerName

Kategorie: Spieler
Ab Engineversion: 4.9.6.0 CR

Beschreibung

Liefert den Namen eines Spielers mit Markup in dessen Spielerfarbe. Wenn die Spielernummer zu keinem gültigen Spieler gehört, wird 0 zurückgegeben. Dadurch kann diese Funktion auch benutzt werden, um zu ermitteln, ob eine Spielernummer gültig ist.

Syntax

string GetTaggedPlayerName (int iPlayer);

Parameter

iPlayer:
Spielernummer des Spieler, dessen Name abgefragt werden soll

Anmerkung

Die Farbe wird so gewählt, dass der Spielername auf dunklem Untergrund lesbar ist. Dunkle Spielerfarben werden dazu aufgehellt.

Beispiel

protected func Activate(pClonk)
  {
  Message("%s hat das Objekt aktiviert!", 0, GetTaggedPlayerName(GetController(pClonk)));
  return(1);
  }
Objektscript: Beim Aktivieren wird eine globale Nachricht ausgegeben, die den Spieler verpetzt.
