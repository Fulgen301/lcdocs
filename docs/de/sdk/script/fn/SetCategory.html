SetCategory

Kategorie: Objekte / Kategorie
Ab Engineversion: 4.6.5.0 CP

Beschreibung

Ändert die Kategorie eines Objekts.

Syntax

bool SetCategory (int iCategory, object pObj);

Parameter

iCategory:
Objektkategorie. Sollte ein gültiger C4D_x()-Wert sein.
pObj:
[opt] Zu veränderndes Objekt. 0 bei lokalem Aufruf.

Anmerkung
