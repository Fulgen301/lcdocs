EffectVar

Kategorie: Effekte
Ab Engineversion: 4.9.5.0 CE

Beschreibung

Liefert eine Referenz auf eine effektlokale Variable. Diese kann zum Lesen und schreiben verwendet werden.

Syntax

&EffectVar (int iVarIndex, object pTarget, int iNumber);

Parameter

iVarIndex:
Index der Variable. Es sind theoretisch beliebig viele Variablen möglich; da aber die Liste beim Abfragen von Variablen außerhalb des erzeugten Bereiches neu erstellt werden muss, sollten hier nicht größere Werte als nötig genommen werden.
pTarget:
[opt] Zielobjekt des Effektes, auf dessen Variablenspeicher zugegriffen wird. 0 für globale Effekte.
iNumber:
Effektnummer. Effektnummern werden an Effekt-Callbackfunktionen übergeben, sind Rückgabewert von AddEffect und lassen sich über GetEffect aus dem Effektnamen herausfinden.

Anmerkung

Für Beispiele und weitere Hinweise siehe Effekte-Dokumentation.
Siehe auch: AddEffect, ChangeEffect, CheckEffect, EffectCall, Effekte-Dokumentation, GetEffect, GetEffectCount, RemoveEffect
